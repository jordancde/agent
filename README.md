# VAPI Agent Creator

A sleek React app that creates AI phone agents using the VAPI API. Enter a description of how your agent should behave, and the app will:

1. Create an AI assistant with a custom system prompt
2. Purchase a phone number
3. Associate the phone number with the assistant
4. Provide a contact card you can add to your phone

## Getting Started

```bash
npm install
npm run dev
```

## Usage

1. Enter your VAPI API key
2. Give your agent a name (optional)
3. Describe how your agent should behave
4. Click "Create Agent"
5. Add the new contact to your phone

## Tech Stack

- React + TypeScript
- Vite
- VAPI API
